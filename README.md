# Ace Attorneydle

Inspired by popular daily games like Wordle, Gamedle, or Loldle, Ace Attorneydle is a guessing game set in the Ace Attorney universe. It features various modes, such as character, silhouette, quote, or case. Players can test their knowledge of Ace Attorney and its spin-offs.
